# Bass-Extractor(ベースライン切り取り機)
ブラウザで流れている音楽から低音域（ベースライン）をリアルタイムに強調して聴くことができるツールです。楽器の練習や耳コピに役立ちます（その他含め）。

使用はこちらから
 (https://akapikumindayo.github.io/Bass-Extractor/)

仕組み・クレジット:
・Web Audio API を使用した、ブラウザ上で完結する安全な設計です。
・Geminiと協力して開発しました。
・ローパスエフェクターに掛けただけなので超雑音が入ります。
・同じ低音域のバスドラムはもちろん入ってきます。

注意・免責事項:
※本ツールには、キャプチャした音声を録音・保存・サーバーへアップロードする機能は一切ありません。
　個人でのリスニングおよび学習・練習用としてお使いください。
